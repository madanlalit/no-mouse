# NoMouse Keyboard Shortcuts

## Activation

| Shortcut | Action |
|----------|--------|
| `⌃ Space` | Activate NoMouse (Control + Space) |
| `Escape` | Deactivate / Cancel |

## Grid Mode

| Shortcut | Action |
|----------|--------|
| Type 2 letters | Jump cursor to grid cell (e.g., `AB`, `XY`) |
| `Tab` | Switch to Flow Mode (if both modes enabled) |
| `Backspace` | Clear first letter / Exit refinement |

## Clicking

| Shortcut | Action |
|----------|--------|
| `Return` | Left click |
| `⇧ Return` | Right click (Shift + Return) |
| `⌃ Return` | Double click (Control + Return) |

## Flow Mode (Arrow Keys / Cursor Movement)

| Shortcut | Action |
|----------|--------|
| `↑ ↓ ← →` | Move cursor |
| `⇧ + Arrow` | Fast movement (Shift + Arrow) |
| `⌥ + Arrow` | Slow/precise movement (Option + Arrow) |
| `⌘ + Arrow` | Scroll instead of move (Command + Arrow) |
| `Tab` | Switch to Grid Mode (if both modes enabled) |

## Drag & Drop

| Shortcut | Action |
|----------|--------|
| `D` | Toggle drag mode (hold mouse button) |

## Scrolling

| Shortcut | Action |
|----------|--------|
| `⌘ ↑` | Scroll up |
| `⌘ ↓` | Scroll down |
| `⌘ ←` | Scroll left |
| `⌘ →` | Scroll right |
| `⇧ + Scroll` | Fast scroll |

---

**Tip:** You can enable Grid Mode, Flow Mode, or both from the menu bar icon.
